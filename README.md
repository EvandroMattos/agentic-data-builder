# 🚀 Agentic Data Builder

An AI-powered agent that generates governed data pipelines and automatically creates GitHub pull requests.

## 🔥 Features

- Generate PySpark pipelines using natural language
- Enforce governance rules (Bronze/Silver, naming, Delta)
- Automatically create GitHub PRs
- Ready for Databricks-style architectures

## 🧠 How it works

1. User describes a pipeline
2. AI generates code following governance rules
3. System creates a PR automatically
